告别linux下查词之苦吧！

Openyoudao是有道字典在linux下的客户端，在取词翻译的基础上，对查询到的信息进行有效的整合。我们将来会开发出更多、更有用的功能。

特性   Features

取词稳定准确

Openyoudao的取词能力和windows平台下的有道词典一样准确，保证在您为单词而头疼的时候及时出现。

抛弃烦人的按钮

精简整体的界面，让功能更直接，更简洁，更有效率。

支持全面

无论是网页，pdf还是终端，只要有不认识的单词，都有Openyoudao。

下载   Download

Openyoudao目前支持 Debian、Archlinux、Fedora等发行版linux安装：

Archlinux

$ sudo yaourt -S openyoudao

Ubuntu

$ 添加sourcelist:

deb http://ppa.launchpad.net/justzx2011/openyoudao/ubuntu precise main

deb-src http://ppa.launchpad.net/justzx2011/openyoudao/ubuntu precise main

$ 或者使用add-apt-repository 命令：

sudo add-apt-repository ppa:justzx2011/openyoudao

然后sudo apt-get update

Debian

$ wget http://blog.anantshri.info/content/uplo ... ory.sh.txt

$ cp add-apt-repository.sh.txt /usr/sbin/add-apt-repository

$ chmod o+x /usr/bin/add-apt-repository

$chown root:root /usr/bin/add-apt-repository

然后你就可以正常使用 add-apt- repository 命令了。

其他版本的方法请访问“图文教程”
